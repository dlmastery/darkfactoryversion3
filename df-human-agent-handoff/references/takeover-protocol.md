# Takeover Protocol

Agent to human:
1. State the decision needed.
2. Provide alternatives and recommendation.
3. List evidence already gathered.
4. List risks and blocked work.
5. Ask for one concrete decision.

Human to agent:
1. Re-read changed files and artifacts.
2. Identify what the human changed.
3. Preserve rationale and constraints.
4. Re-run affected gates.
5. Resume from the smallest safe next step.

Any takeover must leave a handoff record.
