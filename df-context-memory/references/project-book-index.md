# Project Book Index

Maintain a compact index with:

- Current product goal.
- Current scope and out-of-scope.
- Current requirements.
- Current architecture.
- Current decisions.
- Current domain glossary.
- Current test and verification state.
- Current Attractor Run Record, control graph node, work-ledger item, and refinery gate state.
- Current release and operations state.
- Current risks and accepted trade-offs.
- Current predecessor recovery notes and replay status.
- Current owner and next action.

The index should let a fresh human or Codex session resume without reading the full history.
